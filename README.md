# Pixel Art

That's one of the first projects (if not the first) with user interaction that I developed. <br/>
It is possible to paint a board of pixels with random colors that appear with each page reload.
It is also possible to resize the board size.

Page URL: https://imvictorm.github.io/pixel-art/
